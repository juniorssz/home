Operadores matemáticos:


Adição e subtração:

2 + 1
## [1] 3
5 - 3
## [1] 2


Multiplicação e divisão:

3 * 4
## [1] 12
10 / 3
## [1] 3.333333
5 / 0
## [1] Inf
0 / 5
## [1] 0


Resto da divisão:

10 %% 3
## [1] 1


Parte inteira da divisão:

10 %/% 3
## [1] 3


Parte decimal da divisão:

10 / 3 - 10 %/% 3
## [1] 0.3333333


Objetos


Regras para criar os nomes dos objetos:

  • Não pode iniciar com número;
  • Não pode iniciar com símbolos, como ^, !, $, @, +, -, /, ou *.


Principais objetos:

  • Vetores
  • Matrizes
  • Listas
  • Data frames


Principais tipos de dados


Numérico:

n1 <- 5
n1
## [1] 5
n2 <- 2.76
n2
## [1] 2.76
class(n2)
## [1] "numeric"


Texto:

t1 <- "a"
t1
## [1] "a"
t2 <- "fundamentos da linguagem R"
t2
## [1] "fundamentos da linguagem R"
class(t2)
## [1] "character"


Lógico:

l1 <- TRUE
l1
## [1] TRUE
l2 <- FALSE
l2
## [1] FALSE
class(l2)
## [1] "logical"


Data:

d1 <- Sys.Date()
d1
## [1] "2020-04-23"
class(d1)
## [1] "Date"


Data e Hora:

d2 <- Sys.time()
d2
## [1] "2020-04-23 11:58:27 -03"
class(d2)
## [1] "POSIXct" "POSIXt"


O operador c()


v1 <- c(1,2,3)
v1
## [1] 1 2 3
v2 <- c("gato", "cachorro", "macaco")
v2
## [1] "gato"     "cachorro" "macaco"
v3 <- c(TRUE, FALSE, TRUE)
v3
## [1]  TRUE FALSE  TRUE


Gerenciando objetos


Listando os objetos criados:

ls()
##  [1] "d1" "d2" "l1" "l2" "n1" "n2" "t1" "t2" "v1" "v2" "v3"


Removendo um objeto:

rm(l1)
ls()
##  [1] "d1" "d2" "l2" "n1" "n2" "t1" "t2" "v1" "v2" "v3"


Removendo vários objetos:

rm(v1,v2,v3)
ls()
## [1] "d1" "d2" "l2" "n1" "n2" "t1" "t2"


Removendo todos os objetos:

rm(list = ls())
ls()
## character(0)


Pacotes


Instalando o pacote stringr:

install.packages("stringr")


Utilizando a função str_replace() do pacote stringr para substituir um texto:

nomes <- c("maçã", "banana", "laranja", "melancia")
nomes
## [1] "maçã"     "banana"   "laranja"  "melancia"
nomes2 <- stringr::str_replace(string = nomes, pattern = "maçã", replacement = "uva")
nomes2
## [1] "uva"      "banana"   "laranja"  "melancia"


Carregando o pacote stringr:

library(stringr)


Utilizando novamente a função str_replace(), sem o prexixo ::stringr:

nomes3 <- str_replace(string = nomes, pattern = "maçã", replacement = "uva")
nomes3
## [1] "uva"      "banana"   "laranja"  "melancia"
